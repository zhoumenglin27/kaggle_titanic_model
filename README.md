# kaggle_titanic_model
泰坦尼克号生还者预测
Titanic Survival Prediction — Data Science End-to-End Project

项目简介

本项目使用 Kaggle 的 Titanic 数据集，完成一个从 数据探索 → 特征工程 → 模型训练 → 评估 的端到端机器学习流程。

项目目标：预测乘客是否可以在事故中生存。

最终模型准确率：
	•	训练集：0.82
	•	测试集：0.83

项目结构

.
├── titanic_model.ipynb     # 主要代码流程
├── README.md               # 项目说明文档
└── scr                 # 源代码

1. 数据探索（EDA）

主要分析内容包括：
	•	性别与生存率关系（女性生存率显著更高）
	•	舱位等级对生存的影响（Pclass 越高生存率越高）
	•	年龄、票价分布
	•	家庭人数 (FamilySize)
	•	缺失值分布（Age、Cabin 缺失较多）

2. 特征工程（Feature Engineering）

本项目使用了 实验驱动的特征选择方法（A/B 测试思路）：

✔ 使用的主要特征
	•	Sex（性别）
	•	Pclass（舱位等级）
	•	agegroup（年龄分组）
	•	Fare_mean（票单价）
	•	SibSp （兄弟姐妹数）
•   Parch （父母数）
	•	T_name（称呼）
    •	Embarked（登陆港口）
•	fareclass（车票价格等级）

✔ 特征选择方法

对每个特征进行对照实验（加入或移除）
→ 观察模型交叉验证得分变化
→ 保留能提升模型表现的特征

3. 模型训练与评估

尝试了以下模型：
	•	Logistic Regression（最佳）
	•	LDA（最佳）
	•	PCA（效果差）

最终选择 LDA+Logistic Regression 因为它表现稳定、泛化能力强、可解释性好。

最佳准确率：
训练集准确率: 0.8272
测试集准确率: 0.8380

4. 结果与总结
	•	成功构建了端到端机器学习流程
	•	利用实验驱动方法选择特征
	•	得到了泰坦尼克项目常见的高分段结果（0.82–0.84）
	•	掌握了数据清洗、编码、特征工程、模型评估等关键技能
